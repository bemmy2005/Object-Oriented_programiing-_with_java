Unit 2: Introduction to Java

Java Evolution

History: Java was created by James Gosling at Sun Microsystems (now owned by Oracle) and released in 1995.

Key Java Versions Timeline

  • 1995: Java 1.0 - Initial release
  • 1998: Java 1.2 - "Java 2" with Swing GUI
  • 2004: Java 5 - Generics, annotations
  • 2014: Java 8 - Lambda expressions, Stream API
  • 2018: Java 11 - LTS release
  • 2021: Java 17 - Latest LTS release

Key Features

Secure

Object-Oriented

Everything is an object

Automatic Memory

Garbage collection

Java vs C and C++

Feature Java C/C++
Memory Management Automatic (Garbage Collection) Manual (malloc/free, new/delete)
Platform Platform Independent Platform Dependent
Pointers No direct pointer manipulation Extensive pointer usage
Multiple Inheritance Through interfaces only Supported in C++

Java and the Internet

Java was designed with the Internet in mind:

  • Applets: Java programs embedded in web pages (now deprecated)
  • Servlets: Server-side Java components
  • Web Applications: Enterprise-scale web systems
  • Web Services: RESTful APIs and SOAP services

Note: While Applets were once a major feature, they are now deprecated in favor of modern web technologies.
