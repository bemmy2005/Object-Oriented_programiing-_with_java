Unit 7: Arrays, Strings, and Vectors

One-dimensional Arrays

An array is a container object that holds a fixed number of values of a single type.

// Declaration
int[] numbers;

// Initialization
numbers = new int[5];

// Declaration and initialization
int[] scores = {95, 85, 75, 90, 88};

Key Points:

  • Arrays have fixed size once created
  • Array indices start at 0
  • Arrays store elements of the same type
  • Use array.length to get the size

Strings

Strings in Java are immutable objects that represent sequences of characters.

// String creation
String str1 = "Hello";
String str2 = new String("World");

// String concatenation
String greeting = str1 + " " + str2;

Common String Methods:

Method Description
length() Returns string length
charAt(int) Returns character at position
substring(int, int) Returns substring
equals(String) Compares string content

Introduction to Vectors (ArrayList)

While Vector is part of Java's legacy collections, ArrayList is the modern alternative:

import java.util.ArrayList;

// Creating ArrayList
ArrayList<String> fruits = new ArrayList<>();

// Adding elements
fruits.add("Apple");
fruits.add("Banana");

// Accessing elements
String fruit = fruits.get(0);

ArrayList vs. Array:

  • Dynamic size (can grow and shrink)
  • Provides useful methods (add, remove, contains)
  • Stores only objects, not primitives
  • Slightly slower than arrays

Wrapper Classes

Wrapper classes provide object representations of primitive data types:

Primitive Type Wrapper Class
byte Byte
short Short
int Integer
long Long
float Float
double Double
char Character
boolean Boolean 
// Autoboxing (primitive to wrapper)
Integer num = 100;

// Unboxing (wrapper to primitive)
int value = num;

Array and String Example

public class ArrayStringExample {
    public static void main(String[] args) {
        // Array of strings
        String[] names = {"Alice", "Bob", "Charlie"};
        
        // Loop through array
        for (String name : names) {
            // String manipulation
            String greeting = "Hello, " + name.toUpperCase() + "!";
            System.out.println(greeting);
        }
    }
}
