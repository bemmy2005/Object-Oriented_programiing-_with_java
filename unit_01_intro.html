Unit 1: Introduction

Internet Origin & Development

The Internet began as ARPANET in the late 1960s, a U.S. Department of Defense project to create a decentralized communication network.

Key milestones:

  • 1969: First ARPANET connection established
  • 1983: TCP/IP protocols adopted
  • 1989: Tim Berners-Lee proposes the World Wide Web
  • 1993: Mosaic web browser released
  • 1995: Commercial Internet begins

Internet Architecture Framework

The Internet operates on a layered architecture:

  1. Application Layer: HTTP, FTP, SMTP, DNS
  2. Transport Layer: TCP, UDP
  3. Internet Layer: IP, ICMP
  4. Link Layer: Ethernet, Wi-Fi

This layered approach allows for modular development and interoperability.

World Wide Web

The World Wide Web (WWW) is an information system built on top of the Internet infrastructure.

Key components:

  • HTML: HyperText Markup Language for creating web pages
  • HTTP: HyperText Transfer Protocol for transmitting web content
  • URLs: Uniform Resource Locators for addressing web resources
  • Web Browsers: Software to navigate and display web content

Internet vs. World Wide Web

Internet World Wide Web
Global network of networks Information system on the Internet
Infrastructure Service built on infrastructure
Includes email, FTP, etc. Primarily web pages and hyperlinks
Developed in 1960s Developed in 1989-1991

Connection to Java Programming

Java was designed with the Internet in mind, offering platform independence through its "Write Once, Run Anywhere" philosophy. This makes it ideal for developing applications that work across the diverse devices and systems connected to the Internet.
