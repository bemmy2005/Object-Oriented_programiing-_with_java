Unit 13: Graphics and GUI Programming

Introduction to Java GUI

Java provides several libraries for creating graphical user interfaces (GUIs):

AWT

Abstract Window Toolkit

  • Platform-dependent
  • Heavyweight components
  • Basic components

Swing

Built on top of AWT

  • Platform-independent
  • Lightweight components
  • More advanced components

Note: Modern Java applications often use JavaFX instead of Swing, but Swing is still widely used and important to understand.

Basic Swing Components

JFrame

Main window container

JPanel

Container for components

JLabel

Display text or images

JButton

Clickable button

JTextField

JTextArea

Multi-line text input

JCheckBox

JRadioButton

Mutually exclusive options

Event Handling

JButton button = new JButton("Click Me");

// Add action listener
button.addActionListener(new ActionListener() {
    @Override
    public void actionPerformed(ActionEvent e) {
        System.out.println("Button clicked!");
    }
});

Layout Managers

Layout managers control the position and size of components:

BorderLayout

Five regions: North, South, East, West, Center

FlowLayout

Components in a row, wrapping as needed

GridLayout

Equal-sized cells in rows and columns

BoxLayout

Components in a single row or column

// Setting layout
panel.setLayout(new BorderLayout());

// Adding components with constraints
panel.add(button, BorderLayout.NORTH);

Graphics in Java

Java provides classes for drawing graphics:

Class Description
Graphics Basic drawing operations
Graphics2D Advanced drawing with better control
Color RGB color representation
Font Text font styling 
@Override
protected void paintComponent(Graphics g) {
    super.paintComponent(g);
    
    // Cast to Graphics2D for advanced features
    Graphics2D g2d = (Graphics2D) g;
    
    // Set color and draw a rectangle
    g2d.setColor(Color.BLUE);
    g2d.fillRect(50, 50, 100, 80);
    
    // Set font and draw text
    g2d.setFont(new Font("Arial", Font.BOLD, 16));
    g2d.setColor(Color.RED);
    g2d.drawString("Hello, Graphics!", 60, 100);
}

Simple GUI Example

import javax.swing.*;
import java.awt.*;
import java.awt.event.*;

public class SimpleGUI {
    public static void main(String[] args) {
        // Create on Event Dispatch Thread
        SwingUtilities.invokeLater(() -> {
            // Create frame
            JFrame frame = new JFrame("Simple GUI");
            frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
            frame.setSize(300, 200);
            
            // Create components
            JLabel label = new JLabel("Enter your name:");
            JTextField textField = new JTextField(20);
            JButton button = new JButton("Submit");
            
            // Add action listener
            button.addActionListener(e -> {
                String name = textField.getText();
                JOptionPane.showMessageDialog(frame, 
                    "Hello, " + name + "!");
            });
            
            // Create panel and add components
            JPanel panel = new JPanel();
            panel.setLayout(new FlowLayout());
            panel.add(label);
            panel.add(textField);
            panel.add(button);
            
            // Add panel to frame
            frame.add(panel);
            frame.setVisible(true);
        });
    }
}
