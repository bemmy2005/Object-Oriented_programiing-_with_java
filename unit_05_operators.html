Unit 5: Operators and Expressions

Arithmetic Operators

Operator Description Example
+ Addition a + b
- Subtraction a - b
* Multiplication a * b
/ Division a / b
% Modulus (Remainder) a % b 
int a = 10;
int b = 3;

int sum = a + b;        // 13
int difference = a - b;  // 7
int product = a * b;     // 30
int quotient = a / b;    // 3 (integer division)
int remainder = a % b;   // 1

// Division with floating-point result
double result = (double)a / b;  // 3.3333...

Unary Operators

Increment/Decrement

int x = 5;
x++;  // Post-increment: x = 6
++x;  // Pre-increment: x = 7
x--;  // Post-decrement: x = 6
--x;  // Pre-decrement: x = 5

Other Unary

int a = 10;
int b = -a;  // b = -10
boolean flag = true;
boolean not = !flag;  // not = false

Pre vs. Post Increment:

int a = 5;
int b = a++;  // b = 5, then a = 6

int c = 5;
int d = ++c;  // c = 6, then d = 6

Assignment Operators

Simple Assignment

int x = 10;  // Assign 10 to x

Compound Assignment

x += 5;   // Same as: x = x + 5
x -= 3;   // Same as: x = x - 3
x *= 2;   // Same as: x = x * 2
x /= 4;   // Same as: x = x / 4
x %= 3;   // Same as: x = x % 3

Relational Operators

Operator Description Example
== Equal to a == b
!= Not equal to a != b
> Greater than a > b
< Less than a < b
>= Greater than or equal to a >= b
<= Less than or equal to a <= b 
int a = 10;
int b = 20;

boolean isEqual = (a == b);      // false
boolean isNotEqual = (a != b);   // true
boolean isGreater = (a > b);     // false
boolean isLess = (a < b);        // true
boolean isGreaterEq = (a >= b);  // false
boolean isLessEq = (a <= b);     // true

Logical Operators

AND

boolean result = (a > 5) && (b < 30);

True only if both conditions are true

OR

boolean result = (a > 5) || (b < 30);

True if either condition is true

NOT

boolean result = !(a > b);

Inverts the boolean value

Short-circuit Evaluation: In && and || operations, the second condition is only evaluated if necessary.

Bitwise Operators

Bitwise AND, OR, XOR

int a = 5;  // 0101 in binary
int b = 3;  // 0011 in binary

int c = a & b;  // 0001 = 1
int d = a | b;  // 0111 = 7
int e = a ^ b;  // 0110 = 6

Shift Operators

int x = 8;    // 1000 in binary

int y = x << 2;  // 100000 = 32
int z = x >> 1;  // 0100 = 4
int w = x >>> 1; // 0100 = 4

Operator Precedence

Operators are evaluated in a specific order. When in doubt, use parentheses to control the order of evaluation.

Higher precedence:

  1. Postfix operators (expr++, expr--)
  2. Prefix operators (++expr, --expr, +expr, -expr, !, ~)
  3. Multiplicative (*, /, %)
  4. Additive (+, -)
  5. Shift (<<, >>, >>>)

Lower precedence:

  1. Relational (<, >, <=, >=, instanceof)
  2. Equality (==, !=)
  3. Bitwise AND (&)
  4. Bitwise XOR (^)
  5. Bitwise OR (|)
  6. Logical AND (&&)
  7. Logical OR (||)
  8. Assignment (=, +=, etc.)
int result = 10 + 5 * 2;  // 20, not 30 (multiplication has higher precedence)
int result2 = (10 + 5) * 2;  // 30 (parentheses control the order)
